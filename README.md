# Jalesma-CRUD

Actualmente, el proyecto es un sistema web desarrollado en Spring Boot con Thymeleaf que permite gestionar bolsos, marcas y materiales mediante funcionalidades CRUD completas. Los bolsos están relacionados con las marcas mediante una relación *Many-to-One* y con los materiales mediante una relación *Many-to-Many*, lo que permite registrar bolsos con múltiples materiales. El sistema también incluye manejo de eliminación en cascada para marcas asociadas a bolsos y formularios estilizados que adaptan su contenido según si el usuario está creando o editando un registro. Además, se ha personalizado la interfaz agregando un logo cargado desde los recursos estáticos del proyecto.

En su estado actual, la aplicación cuenta con una estructura MVC funcional, persistencia de datos usando JPA/Hibernate, rutas y controladores organizados, y una base sólida para continuar incorporando características como validaciones, filtros de búsqueda, roles de usuario y más opciones de administración.
